# Consultas-SQL-com-JOINs-e-Analise-de-Dados-em-MySQL

Este repositório contém consultas SQL em MySQL aplicadas a um banco de dados de restaurante, utilizando JOINs, filtros, ordenação, agrupamentos e cálculos agregados para análise e organização de dados relacionais. O projeto é voltado para prática em Banco de Dados e Análise de Dados.

## Ferramentas utilizadas

* MySQL
* SQL
* JOIN / LEFT JOIN
* WHERE, ORDER BY, GROUP BY
* Funções agregadas (`COUNT`, `SUM`)
* Modelagem relacional de tabelas
* Ambiente de estudos em Banco de Dados
